# tsv-comparison-tool
tsv比較したい→テキストファイル比較したいになったけど、名前の名残があるブツです。

説明なしで視覚的に使えます

https://azuwata.github.io/tsv-comparison-tool/

## 使用技術
- Frontend: HTML5 + CSS3 + JavaScript (ES6+)
- ファイル処理: File API + FileReader
- データ処理: JavaScript（ライブラリなし）
- UI: CSS Grid + Flexbox
- レスポンシブ: CSS Media Queries
- 出力: Blob API + CSV形式
